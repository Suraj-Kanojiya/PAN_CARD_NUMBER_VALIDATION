# PANCARD_NUMBER_VALIDATION

## Project Description
This project implements **PAN (Permanent Account Number) validation** using **MS SQL Server (T-SQL)**.

The objective is to clean raw PAN data and classify each PAN as **Valid** or **Invalid** based on predefined business and logical validation rules.

All validation criteria, rules, and examples are documented in the **attached PDF** included in this repository.

---

## Process
- PAN data cleaning and normalization
- Validation using T-SQL string functions
- Custom User Defined Functions (UDFs)
- Use of CTEs and Views for structured logic
- Classification of PANs into Valid and Invalid categories

---

## Technologies Used
- MS SQL Server
- T-SQL
